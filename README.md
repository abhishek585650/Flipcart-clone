# 🛒 Flipkart Clone

This is a Flipkart Clone project developed using HTML, CSS, and JavaScript. It replicates the basic functionality and design of an e-commerce website.

## 🚀 Features
- Product listing
- Search functionality
- Add to cart
- Responsive design
- Multiple product pages

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📂 Project Structure
- index.html (Homepage)
- Product pages (clothing, bags, etc.)
- Images and assets
